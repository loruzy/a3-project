# A3 Project

A3 Project is a static multi-page fashion e-commerce website built with HTML, CSS, and vanilla JavaScript. The project includes a home page, catalog, product detail page, blog, about page, contact page, and cart layout.

## Preview

Open `index.html` in a browser to view the website locally.

## Pages

- `index.html` - landing page with hero, featured products, banners, and newsletter section
- `shop.html` - product catalog
- `sproduct.html` - product detail page
- `blog.html` - editorial/blog layout
- `about.html` - brand/about page
- `contact.html` - contact form and map section
- `cart.html` - shopping cart layout

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome icons
- Google Fonts

## Features

- Responsive navigation menu
- Product cards and promotional sections
- Static cart, contact, blog, and product detail layouts
- Mobile-friendly CSS media queries

## Run Locally

No build step is required.

```bash
git clone https://github.com/loruzy/a3-project.git
cd a3-project
```

Then open `index.html` in your browser.
